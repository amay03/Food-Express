# Images Folder

Place your food item images in this folder with the following naming convention:

## Required Images:

### Burgers:
- `burger-classic.jpg` - Classic Burger
- `burger-chicken.jpg` - Chicken Burger  
- `burger-veggie.jpg` - Veggie Burger

### Snacks:
- `fries.jpg` - French Fries
- `onion-rings.jpg` - Onion Rings
- `nachos.jpg` - Loaded Nachos

### Drinks:
- `cola.jpg` - Cola
- `iced-tea.jpg` - Iced Tea
- `milkshake.jpg` - Milkshake

### Desserts:
- `brownie.jpg` - Chocolate Brownie
- `cheesecake.jpg` - Cheesecake
- `ice-cream.jpg` - Ice Cream

## Notes:
- Images will automatically fallback to online images if local files are not found
- Supported formats: JPG, PNG, GIF, WebP
- Recommended size: 600x400px or similar aspect ratio
